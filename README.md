# 3D Runner Game

Простая 3D-раннер игра в стиле Subway Surfers, созданная на Godot Engine.

## Структура проекта

```
/workspace
├── project.godot          # Конфигурация проекта Godot
├── icon.svg               # Иконка проекта
├── README.md              # Этот файл
├── scenes/
│   ├── main_menu.tscn     # Сцена главного меню
│   └── level.tscn         # Сцена игрового уровня
├── scripts/
│   ├── main_menu.gd       # Логика главного меню
│   ├── player.gd          # Логика игрока
│   └── level.gd           # Логика уровня (спавн препятствий и бонусов)
└── assets/                # Папка для ресурсов (текстур, моделей, звуков)
```

## Управление

- **A / Стрелка влево** - Перемещение влево
- **D / Стрелка вправо** - Перемещение вправо
- **Пробел / W** - Прыжок

## Особенности игры

- Автоматический бег персонажа
- Три полосы движения
- Препятствия (красные кубы)
- Бонусы (синие сферы) +50 очков
- Система очков и уровней
- Увеличение скорости с ростом уровня
- Экран Game Over с возможностью перезапуска

## Как запустить

1. Откройте проект в Godot Engine 4.2+
2. Нажмите F5 для запуска игры
3. Или экспортируйте проект для вашей платформы

## Примечания

- Мультиплеер и таблицы лидеров не реализованы (по требованию)
- Все ресурсы генерируются программно (без внешних ассетов)
- Игра полностью функциональна и готова к расширению
